# Typescript/javascript Interview Assignments

## What is this repository for?

This repository is used as a **live coding exercise during a technical interview** for Python developers.
It contains a small set of **practical programming and algorithmic assignments**, each backed by unit tests.

The goal is **not** to trick you or test memorization, but to understand how you:

- Think through problems
- Write readable, correct code
- Communicate while solving problems
- Work collaboratively with another developer

---

## How the interview works

- You will **share your screen** during the interview
- We will work through the assignments **together**
- You are encouraged to **think out loud**
- Ask questions, explain tradeoffs, and discuss alternatives

There are **no trick questions** and no “gotcha” moments.

---

## Rules

- ✅ You may use **Google or any search engine**
- ❌ **AI tools are not permitted**
- ❌ Do not modify test files
- ✅ Focus on correctness and clarity over cleverness
